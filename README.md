# Рефакторинг GUI манипулятора с электроэрозией

### Выполнили студенты группы ИПР-22-1б: Малых Игорь (@faweddd), Петров Мавтей (@yvelyu), Семенов Лев (@chikiryauxgod)

Репозиторий содержит результаты рефакторинга графического пользовательского интерфейса (GUI) системы управления манипулятором электроэрозионной обработки (EDM).
Цель работы — улучшение архитектуры кода, читаемости и сопровождаемости системы.

## Оценка коммитов с помощью LLM

Каждый коммит в истории проекта был автоматически проанализирован на соответствие принципам **SOLID** с использованием трёх больших языковых моделей:

* **gemma2:9b**
* **olmo-3:7b**
* **qwen2.5:7b**

Результаты оценки (баллы и пояснения моделей) сохранены в каталоге `docs/`.

## Структура репозитория

* `src/` — исходный код рефакторинга GUI
* `scripts/` — скрипты для извлечения истории коммитов, нормализации diff, разбиения на чанки, анализа LLM и агрегации результатов
* `docs/` — результаты оценки коммитов и вспомогательные материалы
